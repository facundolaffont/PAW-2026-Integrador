Inteligencia artificial Modelo

Los bots usan Gemini 1.5 Flash para decidir jugadas. Si la API falla o devuelve una jugada inválida, cae a un modo fallback determinista.

Persistencia Persistencia

Persistencia es un singleton que delega jugadores y puntajes al repositorio activo (MySQL si DB_HOST está definido, memoria en caso contrario). Las partidas activas viven siempre en un Map en memoria.

Capa HTTP HTTP

Los handlers exponen rutas Express y delegan en los servicios de aplicación. ManejadorFront renderiza vistas EJS. El middleware requireAuth / requireAuthWeb valida el JWT en cookie.
